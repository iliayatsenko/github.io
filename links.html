Collection of interesting links.

This is my collection of interesting links, mostly tech blogs:

  • Vladimir Khorikov - extremely useful blog about DDD, software architecture and testing.

  • Matthias Noback - my second favorite blog about software architecture, mostly PHP and Symfony related.

  • ploeh blog - blog about serious software construction, with a very academical approach.

  • @hgraca - another good place where to read about DDD and software architecture.

  • Teach Yourself Computer Science - opinionated plan of how to learn computer science. Some parts may be useful.

  • Hexagonal Architecture - interesting guide to hexagonal architecture.

  • Connell W – .NET Developer from the UK - about event-driven architecture, CQRS and other interesting things.

  • CodeOpinion - different articles written by a specialist working at Microsoft.

  • Développeur web freelance - blog related to PHP, Symfony and full-stack web-development.

  • Mistaeks I Hav Made - about TDD, CQRS and a lot of another things which I do not understand.

  • Kenneth Lange - about API design and team-management.

  • Dmitriy Lezhnev - useful blog about development, mostly PHP-related.

  • Alex Bespoyasov - another blog about fundamental things in software development.

  • matthew daly - blog about full-stack web-development.

  • Nuno Maduro - blog of one active Laravel developer.

  • Ted M. Young - about software design.

  • Asmir Mustafic - blog of one of Doctrine contributors.
